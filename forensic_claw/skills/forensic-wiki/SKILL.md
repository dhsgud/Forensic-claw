# Forensic Wiki

Use this skill when analysis results should be preserved as structured markdown notes.

## Use Cases

- Save a final finding for a case or artifact
- Turn a chat conclusion into a reusable note
- Preserve analyst-facing summaries outside the active session

## Guidance

- Keep wiki notes concise, evidence-oriented, and easy to skim.
- Separate observations, interpretation, and remaining questions.
- Prefer durable conclusions over transient tool chatter.
