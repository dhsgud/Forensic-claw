# Memory

Use this skill for durable workspace facts that should survive beyond a single conversation.

## Use Cases

- Preserve repository conventions
- Record analyst preferences that remain stable across sessions
- Save durable case facts that should be recalled later

## Guidance

- Save only information that is likely to matter again.
- Do not store volatile scratch notes as long-term memory.
- Prefer concise, high-signal updates.
