# Summarize

Use this skill when the user needs a shorter view of a large result.

## Use Cases

- Summarize long log output
- Collapse repetitive findings into grouped bullets
- Turn raw tool output into a concise forensic digest

## Guidance

- Preserve key facts, counts, timestamps, and anomalies.
- Remove repetition and low-signal filler.
- Make it easy for the user to decide whether a deeper drill-down is needed.
