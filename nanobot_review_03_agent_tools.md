## 1. `agent/tools/registry.py` (도구 대여소)
나노봇이 쓸 수 있는 모든 도구를 모아놓고 관리하는 도구 대여소 파일이에요.

```python
"""Tool registry for dynamic tool management."""

class ToolRegistry:
    # "여기는 나노봇 전용 도구 대여소입니다!"
    
    def register(self, tool):
        # 새로운 도구를 대여소에 등록해요. (예: 새로 산 웹 브라우저 도구 추가!)
        
    def get_definitions(self):
        # AI(뇌)에게 "나노봇아, 너는 지금 망치(검색)와 드라이버(파일읽기)를 쓸 수 있어" 하고 
        # 도구 사용 설명서를 쫙 뽑아서 가져다주는 함수예요.

    async def execute(self, name, params):
        # AI가 "망치 도구 쓸래! 검색어는 '오늘의 날씨'!" 하고 요청하면, 
        # 여기서 진짜로 그 도구를 꺼내서 쾅쾅 실행시켜 주는 아주 중요한 함수입니다.
        # 만약 에러가 나면? "에러 났으니 다른 방식으로 다시 시도해봐!"라고 안전하게 알려주죠.
```

## 2. `agent/tools/` 내부의 다양한 도구들 (로봇의 손과 발)
이 폴더 안에는 나노봇이 실제로 세상을 만질 수 있게 해주는 각양각색의 도구 파일들이 들어있어요.

### `web.py` (인터넷 검색용 안경)
- 인터넷의 바다를 헤엄치는 브라우저 도구예요.
- **`WebSearchTool`**: "구글이나 Brave 검색 엔진에 키워드를 쳐서 상위 5개 문서 제목을 가져와!"
- **`WebFetchTool`**: "찾은 뉴스 기사 링크에 직접 접속해서 그 안의 글씨를 전부 읽어와!" 기능을 해요.

### `filesystem.py` (컴퓨터 정리 정돈용 손)
- 우리 컴퓨터 안에 있는 폴더나 파일을 만지는 도구들!
- **`ReadFileTool`**: 지정한 텍스트 파일을 눈으로 싹 읽어옵니다.
- **`WriteFileTool` / `EditFileTool`**: 새로운 파일을 쓰거나, 기존 파일의 내용을 수정해 줘요. 코딩할 때 제일 많이 쓰겠죠?
- **`ListDirTool`**: "이 폴더 안에 무슨 파일이 있는지 목록 좀 줘!" 하고 폴더 안을 들여다봅니다.

### `shell.py` (터미널 명령 스위치)
- **`ExecTool`**: 컴퓨터의 '명령 프롬프트(cmd)'나 터미널에 직접 명령어를 타다닥 치는 도구예요. 파이썬 프로그램을 실행시키거나 폴더를 강제로 지울 수도 있는 강력한 무기입니다!

### `message.py` (메신저 무전기)
- **`MessageTool`**: "사용자에게 이 사진 좀 전송해 줘!" 할 때 씁니다. 그냥 대답하는 게 아니라 특별히 외부 채널에 파일이나 미디어를 적극적으로 쏠 때 쓰는 무전기예요.

### `cron.py` (로봇 전용 알람 시계)
- **`CronTool`**: "내일 아침 7시에 나한테 비 오는지 알려줘"라고 할 때, 타이머를 맞춰두는 도구예요.
